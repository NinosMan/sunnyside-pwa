🏖️ SunnySide

A simple “best beach right now”

How SunnySide picks

SunnySide does a nerdy wind + shelter check in the background, then shows only quick labels.

  • Finds nearby beaches/bays/coves from OpenStreetMap.
  • Gets current wind (speed, gust, direction) from Open‑Meteo.
  • Estimates hill/headland shelter by sampling elevations upwind.
  • Scores “how comfy to sit/lie down” using pedestrian wind comfort criteria (Lawson/Davenport).
Nerdy details

Converts 10m wind to “at-the-beach” height using a log wind profile: U(z)=U(10m)*ln(z/z0)/ln(10/z0).

Shelter uses the upwind blocking angle atan(Δh/d) (bigger/closer hills → more shelter).
